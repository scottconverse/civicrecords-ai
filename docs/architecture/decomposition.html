Sub-Project Build Sequence (from master spec section 9)

Sub-Project 1
Foundation
Docker Compose stack, user auth (JWT + 4 roles), hash-chained audit logging, Alembic migrations, admin panel skeleton, install scripts
Sub-Project 2
Ingestion Pipeline
Data source config, fast track parsers (7 formats), LLM track (Gemma 4 multimodal), chunking engine, embedding via nomic-embed-text, Celery workers
Sub-Project 3
RAG Search + UI
Natural language search, hybrid retrieval (semantic + keyword), source attribution, filters, iterative refinement, AI output labeling
Sub-Project 4
Request Tracking
Intake form, search-to-request association, 11-status workflow, document caching, deadline dashboard, response letters, review/approval
Sub-Project 5
Exemption & Compliance
Rules engine (PII regex + keywords), LLM suggestions, per-state config, flag workflow, auditability dashboard, compliance templates
